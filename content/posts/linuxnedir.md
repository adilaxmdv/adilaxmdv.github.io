# Linux 
<img align="right" width="200" height="200" src="https://chrisreeves.co.nz/wp-content/uploads/2014/05/linux-logo-300x300.jpg">
Yaradıcısı Linus Torvalds olan OpenSource(açıq kodlu) adlanan sərbəst paylaşıla bilən əməliyyat sistemidir. Linux termini ilk dəfə 5 oktyabr 1991 ci ildə internetdə bir mesajda işlətmişdi Linus Torvalds.Əslində linux əməliyyat sistemindən çox əməliyyat nüvəsidir. Yəni linux əməliyyat sistemləri onun nüvəsi üzərində yaradılır. Əsl adı GNU/Linux adlanır. Bu nisbətən uzun olduğu üçün hərkəs ona sadəcə Linux deyir.

# Linux Növləri (Distributions)
Linux OpenSource olması və məşhur kernel(nüvə) olmasına görə bir çox istifadəçi ona yeni funksiyalar əlavə edib, dizayn verərək bazara çıxarıb digər istifadəçilərin də əl çatmasına yardım edib. 
### Ən məşhur Linux növləri :
- Ubuntu
- Debian
- Solaris
- CentOs
- OpenSuse
- Elementary Os


# Linux nə üçün istifadə olunur ?
- Linuxun istifadə olunma səbələrindən ən əsası onun Pulsuz bir əməliyyar sistemi olmasıdır. Heçbir lisenziya kimi şeylər gərək duymadan yükləmək olar 
- Digər səbəb isə OpenSource olmasıdır. İstənilən istifadəçi öz zövqünə 
uyğun dəyişiklik edə bilər
- Daha çox developerlar və sistem inzibatçıları tərəfdən istifadə olunan Linux səbəbi Windows-a müqayisədə daha qəliz olmasıdır. Lakin linuxun növlərindən biri olan Ubuntu daha istifadəçı dostudur.Linuxa yeni başlayanlara və başlamaq istəyənlərə məsləhət görülən növlərdən biridir
